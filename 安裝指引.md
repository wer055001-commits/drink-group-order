# VibeCoding 安裝指引

開始之前，你只需要把開發環境裝好。裝好之後，所有程式都會由 AI 幫你完成。

---

## 自動安裝（推薦）

只要一行指令，就能裝好所有東西。

### 步驟

1. 按下鍵盤的 `Win` 鍵，搜尋 **PowerShell**，點擊開啟
2. 輸入以下指令，按 Enter：

```
cd d:\VibeTemplate
.\setup.ps1
```

3. 等待腳本跑完，看到 **✅ 環境安裝完成！** 就代表成功了

### 第一次跑完如果出現黃色警告

這是正常的。剛裝好的軟體需要重新開啟視窗才能使用。

**解法：** 關閉 PowerShell → 重新開啟 → 再執行一次 `.\setup.ps1`，第二次會直接通過。

### 出現紅色錯誤怎麼辦？

腳本會告訴你是哪個軟體裝失敗。請到下方「手動安裝」段落，找到對應的軟體，用瀏覽器下載安裝。

---

## 手動安裝（備用方案）

如果自動安裝失敗，或你的電腦沒有 `winget`，可以手動下載安裝。
每個軟體都是「下載 → 打開 → 一直按下一步」就能裝完。

### 1. Visual Studio Code

- 下載：https://code.visualstudio.com/
- 點擊頁面上最大的 **Download** 按鈕
- 安裝時建議勾選「加入右鍵選單」

### 2. .NET 10 SDK

- 下載：https://dotnet.microsoft.com/download/dotnet/10.0
- 選擇 **SDK** 欄位（左邊那個），點你的作業系統對應的安裝檔
  - 大部分 Windows 電腦選 **Windows x64**

### 3. Node.js

- 下載：https://nodejs.org/
- 選擇 **LTS** 版本（頁面上會標明推薦）

### 裝完怎麼確認成功？

開啟 PowerShell，分別輸入以下三行（每行輸入完按 Enter）：

```
dotnet --version
node --version
code --version
```

每行都有顯示版本號碼就代表成功。如果顯示「找不到指令」，請關閉 PowerShell 重新開啟再試一次。

### VS Code 套件

裝好 VS Code 後，在 PowerShell 輸入：

```
code --install-extension ms-dotnettools.csdevkit
code --install-extension Vue.volar
```

或者打開 VS Code，點左邊的拼圖圖示，搜尋 **C# Dev Kit** 和 **Vue - Official** 並點擊安裝。

## 常見問題

**Q: PowerShell 在哪裡？**
按鍵盤上的 `Win` 鍵，輸入 `PowerShell`，點擊搜尋結果中的「Windows PowerShell」。

**Q: 出現「無法辨識 .\setup.ps1」的錯誤？**
請確認你已經先執行 `cd d:\VibeTemplate` 切換到正確的位置。

**Q: 出現「無法載入，因為這個系統上已停用指令碼執行」？**
在 PowerShell 輸入以下指令，然後再試一次：
```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```
系統會問你是否確定，輸入 `Y` 按 Enter。

**Q: winget 找不到？**
你的 Windows 版本太舊。請用上方「手動安裝」的方式逐一下載安裝即可。

**Q: 公司電腦有權限限制，裝不了？**
請聯繫 IT 部門，請他們協助安裝以下三個軟體：Visual Studio Code、.NET 10 SDK、Node.js LTS。
